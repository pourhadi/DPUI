- (void) loadStylesFromFile: (NSString *)  fileName
replaceExisting: (BOOL)  replaceExisting
liveUpdate: (BOOL)  liveUpdate 

Load a .dpui style file created with the DynUI Editor.

Parameters
fileNameThe name, with extension, of the style file. (i.e., "Style.dpui"). When live-updating, this string must be the absolute path to the file (i.e., /Users/dpourhad/Projects/App/Style.dpui)
replaceExistingReplace the existing styles or append to them. Adding duplicate styles results in undefine behavior
liveUpdateWhen liveUpdate is YES, changes saved to the style file specified in fileName are instantly reflected in the simulator when the app is running (no need to re-compile).
